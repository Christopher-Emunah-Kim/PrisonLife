# 💡 핵심 역할

채굴 Zone은 석탄 그리드를 관리하고, 플레이어와 채굴 인부가 석탄을 채굴하여 자원을 획득하는 구역이다. 채굴 업그레이드를 통해 채굴 방식이 3단계로 발전한다.

---

# 💡 다른 시스템과 관계

| 연관 시스템 | 관계 방향 | 설명 |
|---|---|---|
| Player | 진입 → 채굴 | 채굴 콜라이더 활성화로 그리드 셀 채굴 |
| MiningWorker | 자율 → 채굴 | FSM으로 그리드 순회하며 자동 채굴 |
| ProductionZone | 채굴 → 생산 | 채굴 인부의 자원이 자동으로 자원 버퍼로 날아감 |
| GameBalanceData | 데이터 조회 | 채굴 업그레이드 단계별 수치 참조 |
| UpgradeZone | 업그레이드 | 채굴 방식 업그레이드 Zone 활성화/비활성화 |

# 💡 기능 명세 및 상세 규칙

## 채굴 그리드

- 크기: 가로 8 × 세로 16 그리드
- 각 셀: 1×1 크기, 석탄 메시 배치
- 채굴 완료 셀: 스케일 축소 DOTween 연출 후 `SetActive(false)`
- 리젠: 채굴 후 일정 시간 경과 시 해당 위치에서 자라나는 DOTween 연출과 함께 `SetActive(true)`
- 리젠 시간: `GameBalanceData._miningRegenTime` (데이터 드리븐)

## 채굴 방식 3단계

채굴은 플레이어/인부의 이동 방향 콜라이더가 그리드 셀에 접촉 시 발생한다.

| 단계 | 방식 | 채굴 범위 | 비고 |
|---|---|---|---|
| 1단계 | 곡괭이 | 전방 1칸 (소형 콜라이더) | 초기 상태 |
| 2단계 | 드릴 | 凸 형태 4칸 (전방 1칸 + 그 뒤 좌/중/우 3칸) | 돌출부가 플레이어에서 먼 쪽 |
| 3단계 | 트랙터 | 삽 Collider 접촉 전체 | 콜라이더 크기 SerializeField 노출 |

**각 단계별 수치 (GameBalanceData 참조)**

| 단계 | 채굴 틱 간격 | 백팩 자원 MAX |
|---|---|---|
| 1단계 (곡괭이) | [TBD] | [TBD] |
| 2단계 (드릴) | 1단계 × 1/1.5 | 1단계 × 1.5 |
| 3단계 (트랙터) | 2단계 × 1/2 | 2단계 × 2 |

## 채굴 흐름

1. 채굴 Zone 진입 시 채굴 콜라이더 활성화
2. 이동 방향에 따라 콜라이더가 닿는 그리드 셀 채굴
3. 채굴된 셀: DOTween 스케일 축소 → `SetActive(false)`
4. 백팩 slot[0] 수량 +1
5. MAX 도달 시: 플레이어 머리 위 MAX UI 표시, 추가 채굴 무시
6. 채굴 Zone 이탈 시 채굴 콜라이더 비활성화

## 채굴 업그레이드 Zone

- 위치: 채굴 Zone 입구 근처 고정 위치
- 단계 구조: 단계별로 아이콘/비용 수치만 변경 (위치 고정)

| 단계 | 표시 | 활성화 조건 |
|---|---|---|
| 1단계→2단계 (드릴) | 드릴 아이콘 + 비용 | 첫 판매 완료 시 컷씬과 함께 SetActive(true) |
| 2단계→3단계 (트랙터) | 트랙터 아이콘 + 비용 | 드릴 업그레이드 완료 시 동일 위치 Zone 교체 |

- 업그레이드 완료 시: 돈 빨려들어가는 DOTween 연출 + 백팩 slot[1] 차감 + HUD 소지금 갱신
- 업그레이드 완료 시: 플레이어 채굴 방식 즉시 전환

# 💡 채굴 인부 (MiningWorker) FSM

## 고용 조건

- 채굴 인부 고용 UpgradeZone 완료 시 3명 동시 스폰
- 스폰 위치: 채굴 인부 업그레이드 Zone 근처 (채굴 그리드 입구와 동일)
- 활성화 조건: 드릴 업그레이드 Zone과 동시에 SetActive(true)

## FSM 상태

| 상태 | 동작 |
|---|---|
| Moving | 가장 가까운 채굴 가능 셀(미예약)로 이동 |
| Mining | 해당 셀 채굴 틱 처리 → 완료 시 Moving으로 전환 |

- Idle 상태 없음 (그리드에 항상 리젠된 셀이 존재)
- 목표 셀 선택: 채굴 가능한 셀 중 가장 가까운 셀
- 중복 예약 방지: 각 인부가 `_targetCell` 멤버 보유, 목표 업데이트 시 기존 예약 해제 후 중복 체크
- 이동 방식: Transform 직선 이동 (`BaseCharacter._moveSpeed` 사용)

## 채굴 인부 자원 전달

- 채굴 1개 완료 시 자원 오브젝트가 인부 위치 → 자원 버퍼 위치로 DOTween lerp로 날아가는 연출 (1개당 1회)
- 인부는 연출과 무관하게 즉시 다음 목표 셀로 이동
- 생산 Zone 자원 버퍼 MAX 시: 채굴은 계속하되 자원 전달 연출만 재생 후 버림 (버퍼에 추가 안 됨)
