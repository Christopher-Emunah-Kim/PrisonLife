# 💡 HUD 구성

| 요소 | 위치 | 컴포넌트 | 상세 |
|---|---|---|---|
| 게임 로고 | 좌상단 | Image | 상시 표시 |
| PLAY 버튼 | 로고 하단 | Button | 탭 시 [STORE_URL] 이동. 상시 표시 |
| 돈 아이콘 | 우상단 | Image | 달러 지폐 아이콘 |
| 소지금 표시 | 돈 아이콘 우측 | TextMeshPro | 백팩 slot[1] 수량 실시간 반영 |
| 음소거 버튼 | 좌하단 | Button | 초기 상태 소리 켜짐. 토글 방식 |
| 감옥 카운터 | 감옥 Zone 입구 벽면 | TextMeshPro (World Space) | `현재수용/한도` 형식 |

## HUD 갱신 규칙

- 소지금 TMP: `OnMoneyChanged` 이벤트 구독, 수량 변경 시 즉시 갱신
- 감옥 카운터 TMP: `OnPrisonCountChanged` 이벤트 구독
- 감옥 카운터 색상: 여유 있음=흰색, 만원=빨간색

# 💡 MAX UI (MaxIndicatorUI)

공통 컴포넌트로 Player와 각 Zone에 재사용.

```csharp
public class MaxIndicatorUI : MonoBehaviour
{
    [SerializeField] GameObject _indicator; // World Space Canvas
    public void SetVisible(bool visible) => _indicator.SetActive(visible);
}
```

| 부착 위치 | 호출 주체 | 표시 조건 |
|---|---|---|
| Player 자식 오브젝트 (머리 위) | InventoryComponent | 백팩 슬롯 MAX 도달 시 |
| 각 Zone (Zone 위 고정) | 해당 Zone | 버퍼 MAX 도달 시 |

# 💡 인디케이터 화살표 시스템

## 공통 컴포넌트 (IndicatorArrow)

```csharp
public class IndicatorArrow : MonoBehaviour
{
    [SerializeField] float _bobAmplitude;  // 보빙 진폭 (에디터 노출)
    [SerializeField] float _bobSpeed;      // 보빙 속도 (에디터 노출)
    
    private bool _isTriggered = false;     // 최초 진입 완료 여부
    private Transform _targetZone;         // 가리키는 Zone Transform
}
```

## Type 1 (Zone 고정 화살표)

- Zone Transform에 World Space Canvas로 부착
- 아래를 향하는 화살표 메시/스프라이트
- 위아래 보빙 연출 (Coroutine, `_bobAmplitude`, `_bobSpeed`)
- 소거 조건: `_isTriggered = true` 시 `SetActive(false)` (영구 제거)

## Type 2 (플레이어 주변 방향 화살표)

- 플레이어 주변에 부착, 대상 Zone 방향을 가리키도록 회전
- 활성 조건: 대상 Zone이 카메라 시야 밖에 있을 때
- 비활성 조건: 대상 Zone이 카메라 시야 내에 들어오는 순간 `SetActive(false)`
- 시야 판별: `Camera.WorldToViewportPoint` 기준 0~1 범위 체크

## 튜토리얼 화살표 상태 머신

```
[1] 게임 시작
    → 채굴 Zone Type 1 화살표 ON
    → (채굴 Zone 시야 밖이면) 플레이어 주변 Type 2 ON

[2] 첫 채굴 Zone 진입
    → 채굴 Zone 화살표 영구 제거 (_isTriggered = true)
    → 자원 절반 이상 획득 대기

[3] 자원 절반 이상 획득
    → 자원 투입 Drop Zone Type 1 화살표 ON
    → 채굴 Zone Type 1 화살표 OFF (동시)

[4] 자원 투입 Drop Zone 진입
    → 해당 화살표 영구 제거

[5] 자원 투입 완료
    → 완제품 버퍼 Zone Type 1 화살표 ON

[6] 완제품 버퍼 Zone 진입
    → 해당 화살표 영구 제거

[7] 완제품 픽업 완료
    → 판매 책상 Zone Type 1 화살표 ON

[8] 판매 책상 Zone 진입
    → 해당 화살표 영구 제거

[9] 판매 완료 (첫 판매)
    → 판매금 Zone Type 1 화살표 ON
    → 이후 컷씬으로 업그레이드 Zone 안내 (화살표 없음)

[10] 판매금 Zone 진입
    → 해당 화살표 영구 제거
    → 이후 인디케이터 화살표 없음
```

# 💡 카메라 컨트롤러 (CameraController)

```csharp
public class CameraController : MonoBehaviour
{
    [SerializeField] Vector3 _offset;           // 플레이어 기준 카메라 오프셋
    [SerializeField] float _cutsceneLerpSpeed;  // 컷씬 lerp 속도
    [SerializeField] float _returnDuration;     // 복귀 시간 (기본값 0.8초)
    
    private bool _isCutscenePlaying = false;
}
```

## 평상시 추적

- 플레이어 Transform.position + `_offset` 위치로 즉시 이동 (Lerp 없음)

## 컷씬 처리

1. `_isCutscenePlaying = true` → 플레이어 입력 차단
2. 목표 Zone 위치로 DOTween/Coroutine lerp 이동
3. 일정 시간 대기 (연출 확인)
4. 플레이어 위치로 `_returnDuration`(0.8초)간 lerp 복귀
5. `_isCutscenePlaying = false` → 플레이어 입력 해제

## 구독 이벤트

```csharp
// Awake에서 구독, OnDestroy에서 해제
OnFirstSaleCompleted  → 채굴 업그레이드 Zone 컷씬
OnPrisonFull          → 감옥 확장 Zone 컷씬
```

# 💡 SFX 시스템

- 사운드 로직만 구현, 에셋 없으면 null 체크 후 무시
- 음소거 버튼: `AudioListener.pause` 토글 방식

```csharp
public void PlaySFX(AudioClip clip)
{
    if (clip == null) return; // 에셋 없으면 무시
    _audioSource.PlayOneShot(clip);
}
```

## SFX 카테고리 (에셋 없어도 로직 구현)

| 상황 | SFX |
|---|---|
| 채굴 | 채굴음 |
| 자원 투입 | 투입음 |
| 완제품 생산 | 생산음 |
| 판매 틱 | 판매음 |
| 업그레이드 완료 | 업그레이드음 |
| 돈 수령 | 수령음 |
